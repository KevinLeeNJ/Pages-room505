Btw, I use Arch

Arch Linux 是一种通用操作系统
它是基于 x86-64 架构的 GNU/Linux 发行版 Arch Linux 采用滚动升级模式
初始安装完成的 Arch Linux只是一个基本系统
随后用户可以根据自己的喜好安装需要的软件并配置成符合自己理想的个性化系统。

Pacman & AUR

Pacman是 Arch Linux的一大亮点。它将一个简单的二进制包
格式和易用的构建系统结合了起来。Pacman的目标
是简化对软件包的管理，无论软件包是来自官方软件仓库
还是用户自己创建的软件包。
AUR（Arch 用户存储库）是一个社区驱动的软件存储库
它为 Arch Linux 用户提供超过 85,000 个软件包
带来了很大的便利。

世界上只有两个 Linux 发行版：Arch Linux 与其它
